### 기능

1. 최신 트렌드 영상 확인
2. 키워드 검색
3. 영상 상세 페이지
4. 채널 정보 및 채널에서 업로드한 최신 영상 확인

### 사용 기술

- React with JavaScript
- react-router-dom
- react-query
- react-icons
- axios
- tailwindcss
- timeago.js

### 사용 API

- [Youtube Data API](https://developers.google.com/youtube/v3?hl=ko)
